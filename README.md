# OPC 全国政策导航

> 一键匹配你能申请的 OPC（一人公司）补贴和政策

**在线访问：** https://siuserxiaowei.github.io/opc-policy/

## 功能

- **用户匹配系统** — 填写公司情况，自动匹配可申报的政策
- **结构化政策数据库** — 每条政策拆分为标准字段，方便检索和匹配
- **37座城市覆盖** — 81条政策，130+ OPC社区
- **广州专版详情页** — 含倒计时、搜索、区域筛选、材料清单、深浅主题等交互
- **税务指南** — 三种主体税负对比、税负计算器、区域优惠政策
- **匹配结果导出** — 一键导出匹配报告
